What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

To select an element from the DOM in JavaScript, you can utilize several methods tailored to your needs. One of the simplest is document.getElementById(), which allows you to select a single element by its unique ID. For selecting multiple elements with the same class, document.getElementsByClassName() returns a live HTMLCollection.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where a single event listener is attached to a parent element rather than individual child elements. This approach takes advantage of the event bubbling phase, where events propagate from the target element up to its ancestors in the DOM. By placing the event listener on a parent element.

How do you manipulate an element's attributes and styles using the DOM?

Manipulating an element's attributes and styles in the DOM can be accomplished through various JavaScript methods. To change an element's attributes, you can use the setAttribute() method. For example, element.setAttribute('src', 'image.jpg') updates the src attribute of an image element. You can also use properties like element.id or element.className.
